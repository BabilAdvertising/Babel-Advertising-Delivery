# 📍 نەخشەی نرخی گەیاندنی چاپەمەنی بابل

بەخێربێن بۆ خزمەتگوزاری گەیاندنی **چاپەمەنی بابل**. ئەم سیستەمە بۆ ئاسانکاری ئێوەی کڕیار دروستکراوە تا بتوانن بە شێوەیەکی خێرا و ڕاستەوخۆ تێچووی گەیاندنی کارەکانتان بۆ ناوچە جیاوازەکان بزانن.

### 🔹 چۆن ئەم نەخشەیە بەکاربهێنم؟
بۆ زانینی نرخ و دووری، تەنها ئەم هەنگاوانە پەیڕەو بکە:
1.  لەسەر نەخشەکە، **ناوی شار یان شارۆچکەکەت** بدۆزەرەوە.
2.  **کلیک** لەسەر نیشانەی نرخەکە (Badge) بکە.
3.  سیستەمەکە بە شێوەیەکی ئۆتۆماتیکی **تێچووی گەیاندن**، **دووری بە کیلۆمەتر** و **کاتی خەمڵێنراو**ت بۆ نیشان دەدات.

### 🌟 تایبەتمەندییەکان:
* **نرخی ڕاستەوخۆ:** بینینی نرخی گەیاندن بۆ زیاتر لە ٣٠ ناوچەی جیاواز.
* **خشتەی کاتی:** دەزانیت نزیکەی چەند خولەک گەیاندنەکە دەخایەنێت.
* **بەکارهێنانی ئاسان:** دیزاین کراوە تا لەسەر مۆبایل بە باشترین شێوە کار بکات.

---
**تێبینی:** نرخەکان بەپێی دووری و ناوچەکان دیاریکراون.

**چاپەمەنی بابل - بۆ هەموو جۆرە چاپەمەنی و ڕیکلامێک**
